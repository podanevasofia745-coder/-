# ТрудИИсь

MVP платформы трудоустройства с ИИ — подбор вакансий и кандидатов на основе индивидуальных параметров.

**Слоган:** Работа, которая совпадает с вами

## Возможности MVP

- Лендинг с описанием сервиса
- Два сценария: соискатель и работодатель
- Анкета с навыками, опытом, зарплатой и форматом работы
- ИИ-подбор с процентом совпадения
- ИИ-ассистент в чате для уточнения рекомендаций
- Обучающие материалы

## Запуск

```bash
cd ~/Projects/trudiis
python3 -m pip install -r requirements.txt
python3 app.py
```

Откройте http://localhost:5001 (порт 5000 на macOS часто занят AirPlay)

## Стек

- Python + Flask (backend)
- HTML/CSS/JS (frontend)
- Алгоритм персонализированного подбора (навыки, опыт, город, зарплата, формат)
